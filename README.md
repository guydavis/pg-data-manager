# Postgres Data Manager

Requirements
------------
* Python
* Flask 
* Postgres

Installation
------------
```
git clone https://github.com/guydavis/pg-data-manager.git
cd pg-data-manager/
./pgdm.py
```
visit http://127.0.0.1:5000/
